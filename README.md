# Практикалық жұмыс №1 — Менің бірінші ЖИ-ассистентім

## Жоба туралы

Бұл жоба веб-әзірлеуде қолданылатын жасанды интеллект құралдарын таныстыратын интерактивті веб-бет болып табылады. Бетте үш ЖИ құралы туралы карточкалар, қарапайым чат-интерфейс, қараңғы тақырып және анимациялар бар.

## Қалай іске қосу керек

1. Жобаның барлық файлдарын бір папкаға сақтаңыз.
2. `script.js` файлындағы `YOUR_GROQ_API_KEY_HERE` орнына өз API кілтіңізді енгізіңіз.
3. `index.html` файлын браузерде ашыңыз.

## Пайдаланылған құралдар

- HTML5
- CSS3
- JavaScript
- Groq API
- GitHub Pages

## Мен не үйрендім

- HTML құрылымын дұрыс құруды
- CSS Grid арқылы адаптивті бет жасауды
- JavaScript арқылы DOM элементтерімен жұмыс істеуді
- API-ге fetch арқылы сұраныс жіберуді
- ЖИ құралдарын веб-жобаға қосуды
