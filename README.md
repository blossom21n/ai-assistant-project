# Менің бірінші ЖИ-ассистентім

## Жоба туралы
Бұл жоба веб-әзірлеуде жасанды интеллект құралдарын таныстыруға арналған. Бетте ЖИ құралдары туралы ақпарат, менің тәжірибем және қарапайым чат интерфейсі берілген.

## Қалай іске қосу керек
1. Жобаны жүктеп алыңыз
2. index.html файлын браузерде ашыңыз
3. Немесе VS Code арқылы Live Server қолданыңыз

## Пайдаланылған құралдар
- HTML
- CSS
- JavaScript
- ChatGPT
- GitHub

## Мен не үйрендім
Осы жоба арқылы мен HTML құрылымын жасауды, CSS арқылы адаптивті дизайн құруды, JavaScript көмегімен интерактивтілік қосуды және ЖИ құралдарын веб-жобаға енгізу жолдарын үйрендім.
