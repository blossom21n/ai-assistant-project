## 🚀 Функционалдығы

### Негізгі мүмкіндіктер:
- **8 түрлі ЖИ категориясы** - ең танымал жасанды интеллект құралдарының толық сипаттамасы
- **Интерактивті ЖИ-чат** - Groq API арқылы нақты уақытта сөйлесу (LLaMA 3.1 моделі)
- **Қараңғы/Ашық тақырып** - 2 түрлі түс схемасы (розовый стиль)
- **Анимациялық карточкалар** - скролл кезінде пайда болатын эффект
- **Сұраныс есептегіші** - әр сұранысты тіркейді
- **Толық жауапты дизайн** - барлық құрылғыларға бейімделген

## 💻 Қолданылған технологиялар

### Frontend:
- **HTML5** - семантикалық құрылым
- **CSS3** - заманауи стильдер (Flexbox, Grid, анимациялар)
- **JavaScript (ES6+)** - динамикалық функционал

### API және сыртқы сервистер:
- **Groq Cloud API** - тегін ЖИ-чат үшін
- **LLaMA 3.1-8B-instant** - жылдам және сапалы модель

### Ерекше CSS мүмкіндіктері:
- CSS Grid және Flexbox
- Градиенттер (linear-gradient)
- Анимациялар (@keyframes)
- Backdrop-filter (ш ыны эффектісі)
- Кастомды скроллбар
- Адаптивті медиа-сұраулар

## ⚙️ Орнату және іске қосу

### Жергілікті серверде іске қосу:

```bash
# 1. Репозиторийді клондау
git clone https://github.com/your-username/ai-tools-for-developers.git

# 2. Жоба қалтасына өту
cd ai-tools-for-developers

# 3. Қарапайым HTTP серверін іске қосу (Python)
python -m http.server 8000

# Немесе Node.js арқылы
npx serve .

# 4. Браузерде ашу
http://localhost:8000
